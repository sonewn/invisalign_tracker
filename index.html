인비절라인 착용 트래커

1단계 — Supabase 테이블 생성
Supabase → SQL Editor에서 아래 SQL을 실행하세요. 이미 실행했다면 2단계로 바로 이동하세요.
CREATE TABLE IF NOT EXISTS invisalign_devices ( id SERIAL PRIMARY KEY, device_num INTEGER NOT NULL, start_date DATE NOT NULL, end_date DATE, created_at TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE IF NOT EXISTS invisalign_entries ( id SERIAL PRIMARY KEY, device_id INTEGER REFERENCES invisalign_devices(id) ON DELETE CASCADE, entry_date DATE NOT NULL, start_time TIME NOT NULL, end_time TIME NOT NULL, unworn_minutes INTEGER NOT NULL, created_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE invisalign_devices ENABLE ROW LEVEL SECURITY; ALTER TABLE invisalign_entries ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all_devices" ON invisalign_devices FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "allow_all_entries" ON invisalign_entries FOR ALL USING (true) WITH CHECK (true);
2단계 — API 키 입력
Supabase → Project Settings → API 에서 URL과 anon public key를 복사하세요.
